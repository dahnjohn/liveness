eKYC SDK Demo

This demo shows how to use the eKYC SDK to perform face liveness detection.

How it works:
1. The SDK is loaded from your backend (localhost:4000/sdk/ekyc_with_processid.js)
2. SDK creates a process (if not provided)
3. Opens your React frontend (localhost:5173) in an iframe
4. Your frontend handles the actual liveness detection
5. Results are automatically sent back and iframe closes
6. No success UI shown - direct return to parent page
